# Certificate III in IT - Lab Exercises

This folder contains a collection of programming exercises and practical tasks completed during my **Certificate III in Information Technology** at TAFE NSW.

### Key Learning Modules:
* **Programming Logic:** Implementation of control structures (If-Else) and complex decision-making.
* **Automation:** Utilizing `For` and `While` loops to handle repetitive tasks efficiently.
* **Data Handling:** Practice with Python data structures (Lists, Dictionaries) and basic File I/O operations.
