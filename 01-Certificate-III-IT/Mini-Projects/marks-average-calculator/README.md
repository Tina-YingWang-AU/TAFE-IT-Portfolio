# Marks Average Calculator

This is a Python-based mini-project designed to calculate the sum and average of marks for 5 subjects.

### Features:
- **Input Handling:** Collects marks for 5 subjects and stores them in a list.
- **Data Validation:** Checks if inputs are non-negative integers.
- **Error Handling:** Ensures marks are within the valid range (0-100) and displays custom error messages for invalid inputs.
- **Readability:** Clean code structure with meaningful variable names and f-string formatting.

### Skills Demonstrated:
- Python Lists & Loops
- Data Type Conversion
- Conditional Logic (If-Else)
- Robust Error Handling
